LinguistAgent is a multi-model platform for the automation of linguistic tasks using different LLM strategies (Prompt Engineering, RAG, Fine-tuning). A preliminary tech report can be found here: https://arxiv.org/abs/2602.05493.
The current version uses metaphor identification as an example by adapting Fuoli et al. (2025). The original paper can be found here: https://arxiv.org/pdf/2509.24866.
